# Smart-cargo-ship-handling-using-Edge-based-ai-and-Iot-
Designed and implemented an edge-intelligent IoT system for real-time cargo ship monitoring using ESP32, DS18B20, INA219, and RFID sensors. Enabled low-latency anomaly detection through edge processing and developed a live dashboard using ThingSpeak for continuous monitoring and validation under simulated real-world conditions
